# You can follow this through one of the next options:

## Containers

1. Install Docker
2. Download docker images:
   - `docker pull postgres:15.3`
   - `docker pull dpage/pgadmin4`
3. Turn on images in console
   - `docker compose up -d`
4. To turn off container use
   - `docker compose down`
5. Connect to PgAdmin

## Cloud

1. Create a database PostgreSQL inside cloud service like:

   - Neon.tech
   - Railway
   - etc.

2. Connect to DB in GUI software like:
   - DBeaver
   - TablePlus

I worked with Cloud option creating a DB in Neon.tech and using DBeaver to interact with it
